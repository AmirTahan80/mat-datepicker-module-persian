# Persian Jalali DatePicker for Angular Material


### Angular persian datepicker with awesome features!


```bash
$ yarn install
$ yarn build:lib
$ ng serve
```

Here are instructions how this demo was created.


#### Install

Using `ng-packagr`:

```bash
$ ng new ng-packaged
$ yarn add --dev ng-packagr
```

#### Build

Now, build your library:

```bash
$ yarn build:lib
```
